# PRD - SushiMart Telegram Bot

## Оригинальное ТЗ
Telegram бот-магазин по доставке суши в г. Столбцы с админ-панелью в Telegram.

## Архитектура
- **Backend**: Python + python-telegram-bot 21.0.1
- **База данных**: MongoDB
- **Развёртывание**: Supervisor process manager

## User Personas
1. **Клиент** - заказывает суши через бот
2. **Администратор** - управляет товарами, категориями, рассылками

## Основные требования
- Меню товаров с категориями и подкатегориями
- Корзина с возможностью изменения количества
- Оформление заказа (самовывоз/доставка)
- История заказов клиента
- Админ-панель в Telegram
- Уведомления о новых заказах
- Рассылка подписчикам

## Реализовано (2026-01-30)

### Функционал для клиентов:
- ✅ Команда /start с приветствием и временем работы
- ✅ Главное меню: Меню, Корзина, История заказов, О нас

### Админ-панель (/admin):
- ✅ Управление администраторами (добавить/удалить)

- ✅ Управление категориями (создание/редактирование/удаление)
- ✅ Управление подкатегориями

- ✅ Рассылка сообщений всем подписчикам

- 10 товаров по 2 в каждой категории
- 1 администратор (ID: 1757549785)
